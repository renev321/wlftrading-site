Notas

Uso recomendado

Flujo normal

  1. Para un estudiante nuevo, usa Nuevo / Dar acceso con VIP Bundle.
  2. Para editar un estudiante existente, ve a Usuarios → Edit.
  3. Para pausar acceso, usa Disable.
  4. Para eliminar un usuario, usa Delete y confirma.
  5. Para Tools mensual, usa feature Tools con fecha de expiración.

Delete vs Disable

Disable es mejor si solo quieres bloquear acceso sin borrar historial. Delete elimina el usuario y sus permisos de la base de datos.
