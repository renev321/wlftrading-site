Cómo usar este panel

Flujo recomendado

  1. El usuario paga el curso.
  2. Agregas su Gmail usando VIP Bundle.
  3. Si luego paga herramientas mensuales, le das acceso a Tools con fecha de expiración.
  4. Si quieres otro admin, primero confirma bien el email y luego asigna Admin Panel.

Importante

Este panel necesita endpoints backend para leer/escribir en D1. Si el backend no está conectado, verás un error de API. En ese caso, aplica también el patch del Worker incluido en este paquete.
